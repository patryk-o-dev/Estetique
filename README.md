# Esthétique – Beauty Salon

Esthétique is a modern, elegant SPA website for a local beauty salon. The project is based on React, TypeScript, and Vite.

## Features

- Single Page Application (SPA)
- Photo gallery and slider
- Appointment booking form
- Customer testimonials section
- Services and pricing
- Special offers
- Embedded Google Maps widget (API)
- Responsive design

## Installation

1. Clone the repository:
   ```
   gh repo clone patryk-o-dev/Estetique
   ```
2. Install dependencies:
   ```
   yarn install
   ```
3. Run the project locally:
   ```
   yarn dev
   ```

## Technologies

- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [SCSS](https://sass-lang.com/)
- Yarn

## Directory structure

- `src/components` – page components (e.g. Home, Gallery, Booking)
- `src/assets` – images, fonts, icons
- `src/styles` – global styles and variables
- `public` – public files
